Software

Software is a generic term for custom or commercial code, operating system utilities, open-source software, or other tools used to conduct behavior modeled in ATT&CK. Some instances of software have multiple names associated with the same instance due to various organizations tracking the same set of software by different names. The team makes a best effort to track overlaps between names based on publicly reported associations, which are designated as “Associated Software” on each page (formerly labeled “Aliases”), because we believe these overlaps are useful for analyst awareness.

Software entries include publicly reported technique use or capability to use a technique and may be mapped to Groups who have been reported to use that Software. The information provided does not represent all possible technique use by a piece of Software, but rather a subset that is available solely through open source reporting.

  • Tool - Commercial, open-source, built-in, or publicly available software that could be used by a defender, pen tester, red teamer, or an adversary. This category includes both software that generally is not found on an enterprise system as well as software generally available as part of an operating system that is already present in an environment. Examples include PsExec, Metasploit, Mimikatz, as well as Windows utilities such as Net, netstat, Tasklist, etc.
  • Malware - Commercial, custom closed source, or open source software intended to be used for malicious purposes by adversaries. Examples include PlugX, CHOPSTICK, etc.

S0009 Дипломатия и Правительство

Инструмент внешнеполитического и государственного давления. Включает в себя переговорные процессы, создание правовых ловушек, введение финансовых санкций и транспортных блокад.

S0016 Имперские Коллегии

Созданы в 1717–1720 гг. Отраслевые ведомства, которые начали дробить управление, отбирая контроль над финансами, судами и армией у гетмана.

  • Военная коллегия забирала управление казацкими полками.
  • Коллегия иностранных дел обрезала любые международные контакты Гетманщины.
  • Камер-коллегия устанавливала общеимперские стандарты сборов и пошлин.
S0015 Малороссийская коллегия

Дата основания: 1722 год (Указ Петра I).
Даты существования: 1722–1727, 1764–1786
Тип инструмента: Постоянный орган колониальной администрации на местах.

Первый специализированный орган прямого имперского управления, размещенный непосредственно на территории Гетманщины (г. Глухов). В отличие от центральных приказов, Коллегия функционировала как «правительство над правительством», осуществляя повседневный надзор за гетманом и казацкой старшиной, постепенно перехватывая их функции.

Создана сразу после смерти Ивана Скоропадского для предотвращения избрания нового гетмана. Коллегия стала инструментом трансформации договорной автономии в обычную провинцию. Существовала в двух составах (1722–1727 и 1764–1786), каждый раз знаменуя собой периоды ликвидации гетманской власти.
S0011 Московский патриархат

Лояльные империи религиозные институты, используемые для идеологической легитимизации.
S0012 Оккупационная администрация

Внешние или искусственно созданные лояльные органы управления, заменяющие или дублирующие локальную легитимную власть.
S0014 Правительствующий сенат

Дата основания: 1711 год (Петр I).
Даты существования: 1711 — 1917 гг.
Тип инструмента: Высший орган государственного управления и законодательства.

Центральный «мозг» империи, концентрирующий в себе административную, судебную и законотворческую власть. В рамках колониальных практик Сенат выступал высшей инстанцией, которая легитимировала демонтаж автономий через издание указов, отменяющих локальное право.

Создан для замещения Боярской думы и Ближней канцелярии. С 1722 года Сенат официально руководил деятельностью коллегий и был инструментом превращения «союзных территорий» в регулярные провинции империи.
S0010 Регулярная армия

Вооруженные силы под прямым государственным управлением.
S0017 Тайная полиция и Спецслужбы

Инструменты скрытого вмешательства, подкупа элит, организации прокси-войн, дознания и институционального террора против несогласных. Исторические инкарнации: Преображенский приказ, Секретная канцелярия, ФСК и ФСБ России.
S0013 Царская канцелярия

Период активности: 1699–1711 гг. (как Ближняя канцелярия); функционально — с середины XVII века через систему Приказов.
Тип инструмента: Личный административный аппарат монарха и центральные ведомства.

Переходный инструмент от средневекового «челобитья» к регулярному имперскому диктату. В рамках колониальных практик Канцелярия обеспечивала юридическую базу для поглощения автономий, переводя договорные отношения в плоскость односторонних монарших указов. Она служила «фабрикой смыслов», где создавались формулировки о «вечном подданстве» и «измене».

Включает в себя Малороссийский приказ и Ближнюю канцелярию Петра I. Этот инструмент позволил монарху оперативно управлять Гетманщиной в обход громоздкой Боярской думы, подготавливая почву для создания Сената (S0014) и окончательной унификации империи.
