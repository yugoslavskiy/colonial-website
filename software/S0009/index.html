Дипломатия и Правительство

Инструмент внешнеполитического и государственного давления. Включает в себя переговорные процессы, создание правовых ловушек, введение финансовых санкций и транспортных блокад.

Techniques Used

Domain ID Name Use
T0107 Транспортная блокада

Полное прекращение железнодорожного, автомобильного и воздушного сообщения с республикой, отмена авиарейсов для физической изоляции региона [1].
T0101 Цензура

Создание Временного информационного центра для цензуры и физическое выдворение независимых репортеров из зоны боевых действий [2].
T0108 Экономическая блокада

В апреле 1992 года правительство России ввело жесткие экономические и финансовые санкции, включая блокировку перевода денежных средств предприятиям и частным лицам [1].
